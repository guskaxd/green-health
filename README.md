# Saúde Verde

Saúde Verde é um aplicativo desenvolvido para fornecer informações sobre plantas medicinais e seus benefícios, além de permitir que os usuários pesquisem por ervas e chás medicinais de forma prática e intuitiva.

# Funcionalidades

Pesquisa de Ervas e Chás: Encontre informações detalhadas sobre plantas medicinais, seus usos, preparos e contraindicações.
Favoritos: Salve as suas plantas favoritas para acessá-las facilmente.
Plantas Medicinais: Conheça uma variedade de plantas e seus benefícios à saúde.

Design Moderno: Interface amigável e responsiva para uma melhor experiência do usuário.
